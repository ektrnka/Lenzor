# Настройка Firebase для входа и синхронизации

Чтобы включить вход через Google и синхронизацию данных между устройствами, выполните следующие шаги.

---

## Шаг 1: Создать проект Firebase

1. Откройте [Firebase Console](https://console.firebase.google.com/)
2. Нажмите **«Создать проект»** (или выберите существующий)
3. Укажите название проекта (например, «Линзор»)
4. При необходимости отключите Google Analytics
5. Нажмите **«Создать проект»**

---

## Шаг 2: Добавить Android-приложение

1. В проекте нажмите **«Добавить приложение»** → иконка Android
2. Укажите **ID пакета**: `com.example.my_button_app_new`
3. Пропустите шаги с nickname и SHA-1 (можно добавить позже)
4. Нажмите **«Зарегистрировать приложение»**
5. Скачайте файл **`google-services.json`**
6. **Замените** файл в папке `android/app/google-services.json` (сейчас там заглушка)

---

## Шаг 3: Включить аутентификацию

1. В меню слева выберите **«Сборка»** → **«Authentication»**
2. Нажмите **«Начать»**
3. Во вкладке **«Sign-in method»** включите **«Google»**
4. Укажите публичное имя проекта и email поддержки
5. Нажмите **«Сохранить»**

---

## Шаг 4: Создать базу Firestore

1. В меню слева выберите **«Сборка»** → **«Firestore Database»**
2. Нажмите **«Создать базу данных»**
3. Выберите режим **«Начать в тестовом режиме»** (для разработки)
4. Выберите регион (например, `europe-west1`)
5. Нажмите **«Включить»**

---

## Шаг 5: Сгенерировать firebase_options.dart

1. Установите Firebase CLI (если ещё не установлен):
   ```bash
   npm install -g firebase-tools
   ```

2. Установите FlutterFire CLI:
   ```bash
   dart pub global activate flutterfire_cli
   ```

3. Войдите в Firebase:
   ```bash
   firebase login
   ```

4. В корне проекта выполните:
   ```bash
   dart run flutterfire_cli:flutterfire configure
   ```

5. Выберите созданный проект и платформу Android
6. Команда создаст/обновит файл `lib/firebase_options.dart`

---

## Шаг 6: Правила безопасности Firestore

В **Firestore** → **Правила** добавьте (для разработки):

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
```

Это разрешает пользователю читать и писать только свои данные.

---

## Шаг 7: Добавить SHA-1 (для релиз-сборок)

Для отладки SHA-1 не обязателен. Для релиза:

1. Получите SHA-1:
   ```bash
   cd android && ./gradlew signingReport
   ```

2. В Firebase Console: **Project Settings** → **Your apps** → Android-приложение → **Add fingerprint**
3. Вставьте SHA-1 и сохраните

---

## Шаг 8: Проверка

После настройки:

1. Запустите приложение: `flutter run`
2. При первом запуске должен появиться экран с кнопкой **«Войти через Google»**
3. Нажмите её и войдите через Google-аккаунт
4. В профиле должна отображаться информация об аккаунте и кнопка **«Синхронизировать»**

---

## Без Firebase

Приложение работает и без Firebase: можно нажать **«Продолжить без регистрации»** и пользоваться всеми функциями локально. Вход и синхронизация будут недоступны до настройки Firebase.
